== Kubernetes

=== Tools

{stern}::
Kubernetes utility to easily see colored logs from multiple containers.

{kubectx}::
Small utility to switch through clusters and namespaces smoothly.

{kubectl-tree}::
Can present dependency trees of Kubernetes resources.
